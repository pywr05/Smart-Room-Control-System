# Smart-Room-Control-System
This project is an Arduino-based smart room automation system.

# Features
- Automatic light control using LDR sensor
- Manually control by using keypad
- Fan control based on temperature and humidity
- Monitoring display
- Uses TIP120 transistor and 1N5819 diode
- Energy saving simulation

# Technologies
- Arduino
- Embedded C
- Sensors (LDR, DHT11)
- LCD display
